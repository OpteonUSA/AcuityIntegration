AcuityIntegration - Automation Workflow

Automation Overview

What this automation does: Orders PDR (Property Data Report) and PDC (Property Data Collection) alternative valuation products through Acuity (ClearValue Consulting) and returns completed reports to JaroDesk. Operates as a child flow within the AlternativeProductsRouter.

Key Features: XML-based API integration with Acuity | Asynchronous two-flow design (outbound order + inbound report) | Base64 PDF extraction and upload | Automated JaroDesk order tagging and delivery | Order mapping for async tracking

FLOW 1: Outbound (JaroDesk → Acuity)
1
Trigger: AlternativeProductsRouter Dispatch
Parent router receives JaroDesk webhook for a PDR/PDC product and calls this child flow with order details.
AlternativeProductsRouter JaroDesk Webhook
↓
2
Get JaroDesk Order Details
Retrieve full order data from JaroDesk API (property address, loan info, borrower contacts, product type).
JaroDesk GET /v1/order Token Cache Pattern
↓
3
Build AcuityOrder XML
Map JaroDesk fields to Acuity XML schema: loan type/purpose, subject property, contacts, product code. Use JaroDesk order ID as PartnerReferenceNumber.
Acuity XML Format Null-Safe Access
↓
4
POST to Acuity & Handle Acknowledgement
Send AcuityOrder XML via HTTPS POST with Basic Auth header. Parse synchronous AcuityAcknowledgement response (Success/Error). Store order mapping for async tracking.
Acuity Basic Auth SharePoint Logging
ASYNCHRONOUS GAP — Hours or days pass while Acuity fulfills the order
FLOW 2: Inbound (Acuity → JaroDesk)
5
Trigger: Acuity HTTP POST
Acuity POSTs an AcuityReport XML message to our Power Automate HTTP trigger endpoint when the completed report is ready.
HTTP Trigger Endpoint
↓
6
Parse XML & Extract Document
Parse AcuityReport XML. Extract base64-encoded PDF from Document element. Look up JaroDesk order ID using PartnerReferenceNumber from order mapping.
XML Parsing Base64 Decode
↓
7
Upload PDF to JaroDesk
Upload completed report to JaroDesk order using 3-step file upload pattern: get signed URL, upload to S3, attach to order.
3-Step File Upload JaroDesk File API
↓
8
Tag, Deliver & Notify
Add "Automation" tag to JaroDesk order. Trigger delivery workflow. Log completion to SharePoint. Send Teams notification.
Order Tagging SharePoint Logging Teams Notification
